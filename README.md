# SwiftTasks v1.5.0 - Release Notes

## 🚀 New Features

- **Categories**: You can now add custom categories to your tasks and subtasks! As many as you would like!
- **Filtering**: With the addition of categories, you can now filter your tasks by the main task category!

## 🐛 Bug Fixes

- **Subtasks not expanding**: Fixed an issue where Subtasks were not expanding when clicked
- **Dark Mode**: Fixed an issue where dark mode was incorrectly applying to some components
- **Navbar**:  Fixed an issue causing the navbar styling to become compact when in smaller screens
- **Download for Windows**: Fixed an issue where the download button was showing inside the desktop app
