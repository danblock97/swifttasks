# SwiftTasks v1.6.0 - Release Notes

## 🚀 New Features

- **Search Bar**: We have now implemented a search functionality to be able to find tasks more easily!
- **Improved Filtering**: Choose between categories, due dates and statuses when filtering to find the tasks you really need!
- **Recurring Tasks**: Ever needed to set a task to re occur? Now you can, select from daily, monthly or a custom number of days and let the app do the rest!
- **Window Size**: SwiftTasks will now remember the size of the window you left it at when closing the app so that you wont need to keep resizing!
